IRC/ERC commands

Register your nick name

/msg nickserv register passwd MailAddress (you can type /msg nickserv help for help)

/msg nickserv set password newpasswd (you can type /msg nickserv help set for help, and you can also use this to change email or hidden your info)

Send message to jianlee but can view by everyone

jianlee: msg (you can use tab to finish the name)

Basic IRC commands

/nick newname : change name

/names channalname: 查看当前频道的所有用户

/whois nickname: 查看nickname 信息

/whoami :查看自己

/who ipaddress: 查看从该IP登陆的所有用户

/who channelname: 显示此频道的人

/who * :长格式显示此频道的人

/join #roomname

/part #rootname reason

/quit reaseon

/away reason

/invite nicknanm #房间名

/kick #房间名 nickname reason

/topic #房间名 房间主题

/msg nickname msg : do not open a new window

/query nickname: open a new window

/ignore nickname: ignore a person

/list : see all the channal and topic in the server

/list #ubuntu-cn: 列出该频道的信息而非其成员

/flush 终止当前命令的输出操作

/mode +(-)i :锁住聊天室

/mode +(-)o: 设定管理员权限

/knock #roomname 要求进入私人聊天室

/invite

Very useful

/msg nickserv ghost nick pass : 杀掉你本人进程中断而停留服务器的名字或别人侵犯你的名字专用权时使用。

/msg nickserv info lnxfei: 查看lnxfei注册时候的信息

ERC快捷键:

C-c C-n: /names

C-c C-j: /join

C-c C-b: change channel

C-c C-p: /leave

How to register IRC username and nickname

username和nickname区别: username 是用来登陆的，一个irc服务器只能有一个

nickname相当于昵称，一个服务器可以有多个，但是nickname也可以注册的, 如果你已经注册了一个nickname:lkong, 别人照样可以用lkong来登陆，但是你可以把该nickname release掉， 这样你就可以登陆了

注册username

/msg userserv register lkong password lkong@xxxx.com

<进入邮箱确认后>

/msg userserv activate lkong token

/msg userserv login lkong password

/msg userserv logout

/msg userserv info lkong

Others see: /msg userserv help

注册nickname

/nick lkong2

/msg nickserv register <这样你就register lkong2了>

/msg nickserv drop lkong2

/msg nickserv regain lkong

/msg nickserv release lkong

Others see: /msg nickserv help
